# Seminarski rad I – Kvalitet podataka

Ovaj repozitorijum sadrži teorijski i praktični deo seminarskog rada
na temu kvaliteta podataka u mašinskom učenju.

## Sadržaj
- teorija/ – teorijski deo rada (DOCX i PDF)
- prakticni/ – Jupyter notebook sa praktičnim primerima

## Tehnologije
- Python
- scikit-learn
- pandas
- numpy
- matplotlib
